2) Paragraph - p

3) Horizontal - hr

7) Image tag - img

8) List - Ordered list and Unordered list

9) Tables - tr, th and td

11) Media Elements - Video and audio

CSS - Cascading Style Sheet
==> It is used to style and design a web page
==> CSS Works Together with HTML5
==> Change the Text, color and background
==> To set the image and design a layout
==> Adding a Animation and Transition 

1) Types of CSS
i) Inline CSS
ii) Internal CSS
iii) External CSS
Inline > internal > External

2) CSS Synatax
Selector{
    property: value;
}

3) Types of Selector
i) Elements Selector - (h1) (h2) (p)
ii) Class Selector - (.) - Multiple - No of times we can use the same class name
iii) id Selector - (#) - Unique - Only one time we can use the sdame id name
iv) Universal Selector - (*) - Give some style, it will apply into the entire site
v) Group Selector - Class or id Selector contains different tags, in that different tags we want to Change the same style means we can go with Group Selector

4) Colors and background
===> Colors - name, RGB, Hex

===> background
    background-image: url(./Assets/images.jpg);
    background-repeat: no-repeat;
    background-size: cover;
    background-position: center;

6) Box Modeling
---> content
---> Padding - around the content is called Padding
---> Border - around the padding is called Border
---> Margin - around the Border is called Margin  

7) Position
i) Static (default)
ii) Relative - using position:Relative to be set the limit like left,right,top,bottom
iii) Absolute - Absolute can search the non-static parent, once reach the non-static parent it will act as a same Relative, otherwise it will seaech one more ....... there is no non-static parent means it will directly go to the root(body) of the website/webpage...
iv) Fixed - to be fixed to the place
v) Sticky - using top:0, once reach the top:0 it will act as a same fixed

8) Display - block(default), Inline, Inline-block, flex, grid and none

       flex                             grid
  One dimensional layout          Two dimensional layout
  Row Or Column                   Row And Column 
   Ex: Navbar, cards              Ex: section, dashboard

=====> flex
i) Using Display: flex  (Column into row)   
ii) Flex direction - row, row-reverse, Column, column-reverse
iii) justify-content (X) - start, center, end, space-around, space-evenly, space-between
iv) Align-items (Y) (set the height) - Start, center, end
v) gap - to be set the values
vi) flex-wrap - nowrap, wrap, wrap-reverse
vii) align-content (Y) (set the height) (flex-wrap is enable) - start, center, end, space-around, space-evenly, space-between
viii) flex on items - flex 1,2,3
ix) order - Change the visual order of the property
